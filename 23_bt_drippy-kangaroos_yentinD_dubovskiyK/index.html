HTML tags can reference multiple CSS class stylings like so: "class2 class1", where the two class names are separated by spaces. You can also use preset nice classes thanks to Bootstrap.

If the two classes have conflicting styles like so: .class1 { font-size: 50px; } .class2 { font-size: 5px; }

Then the first class in chronological order of being listed in the class="" string takes precedence. Here it would be class2.
